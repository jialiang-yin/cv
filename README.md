# leon-cv

Minimal one-page resume website for GitHub Pages.

## Stack
- HTML
- CSS
- Vanilla JavaScript

No frameworks, no build tools, and no dependencies are required.

## Files
- `index.html` - page structure and content
- `styles.css` - layout, typography, colors, and responsive rules
- `main.js` - small navigation enhancements
- `AGENTS.md` - repository contribution rules

## Local Preview
Open `index.html` directly in a browser.

## GitHub Pages Deployment
Publish the repository with GitHub Pages from the default branch root. Since the site is fully static, no extra build or deployment configuration is needed.

## Customization
Replace the sample content in `index.html` with final biography, work history, project details, and contact links. Keep the site simple, accessible, and production-ready.
